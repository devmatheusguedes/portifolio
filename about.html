São essas tecnologias:

Java:
Por ser uma linguagem amplamente utilizada no mundo, e isso se faz por varios motivos ja citados, essa linguagem é multiplataforma, é robusta em segurança, escalabilidade e potabilidade. Se tornando assim, uma das principais escolhas de grandes empresas quando pensam em ter uma grande taxa de sucesso no meio digital para construção de APIs, seja no mobile, web ou desktop.
Spring:
Uso o spring, pois é um framework que da suporte à linguagem java para contrução de APIs web, esse framework impulsiona e muito no desenvolvimento, abrangendo todas as caracteristicas que são cruciais em uma boa API, que são segurança, escalabilidade e desempenho.
PostgreSQL:
Uma das opções que uso de gerenciamento de banco de dados, dentre varias ótimas opções no mercado, é o banco de dados relacional PostgreSQL, pois me fornece as ferramentas de trabalho de que preciso com muita simplicidade e objetividade.
